Project Overview

  • An autonomous rover is a smart vehicle that can navigate on its own without human control. It works by sensing its environment, making decisions, moving safely, and sharing real-time information.

  • The goal of this project is to develop a four-wheeled autonomous rover that navigates independently using a vision-based system.

  • It features intelligent obstacle avoidance with ultrasonic sensors for safe movement and streams real-time video and telemetry to a web interface for monitoring and Analysis..

Technology Stack

Controller

Raspberry Pi 4 and Arduino are used as the main controllers. Raspberry Pi handles image processing, web server, and AI tasks, while Arduino manages motor control and sensor interfacing.

Camera

Raspberry Pi Camera Module V2 is used for real-time video capture. It provides high-quality live streaming for monitoring and vision-based obstacle detection.

Sensors

Ultrasonic sensors are used to measure distance and detect obstacles. They help the rover avoid collisions and navigate safely in different environments.

Software

Python is used for backend control and image processing with OpenCV. HTML, CSS, and JavaScript are used to build the web interface for live monitoring and control. .
